# QuizFlex FE Mock

Vue 3 + Vite + Tailwind UI prototype for QuizFlex.

## Run

```bash
npm install --registry=https://registry.npmjs.org/
npm run dev
```

## Included pages

- User: `/`, `/quiz/:id`, `/join-room`, `/results`, `/upgrade`
- Auth: `/login`, `/register`, `/forgot-password`
- Admin/VIP mock: `/admin`, `/admin/questions`, `/admin/questions/create`, `/admin/questions/ai`, `/admin/rooms`, `/admin/reports`, `/admin/payments`, `/admin/users`, `/admin/settings`

## Notes

This is FE-only mock data. No database or backend is connected yet.
